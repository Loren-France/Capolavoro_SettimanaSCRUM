Panoramica del Sistema

Il Sistema di Assistenza Clienti AI Morato Group è una soluzione integrata che combina intelligenza artificiale con un'interfaccia moderna per fornire supporto automatico 24/7 sui prodotti panificati.

🎯 Obiettivi Principali
  • Supporto clienti immediato e continuativo
  • Risposte coerenti basate sul database aziendale
  • Riduzione del carico sul servizio clienti tradizionale
  • Experience utente moderna e intuitiva

Componenti Chiave

1
Interfaccia Chatbot
HTML/CSS/JS con design responsivo e animazioni moderne
2
Server Backend
Node.js + Express per la logica applicativa e API
3
Motore AI
LM Studio con modello Llama per risposte intelligenti
4
Knowledge Base
Database CSV con domande/risposte strutturate

Architettura Tecnica

Frontend (Client-side)

HTML5/CSS3/JavaScript - Interfaccia moderna senza framework con design responsivo e animazioni avanzate.

Struttura Directory 
morato-assistenza/
├── index.html          # Interfaccia chatbot principale
├── server.js          # Server backend Node.js
├── dataset.csv        # Database domande/risposte
├── package.json       # Dipendenze Node.js
└── node_modules/      # Moduli installati

Backend (Server-side)

Stack Tecnologico
  • Node.js + Express.js: Server web e API endpoints
  • LM Studio SDK: Integrazione con il modello AI locale
  • CSV Parse: Lettura del database strutturato
  • String Similarity: Algoritmo di matching per domande simili

Sistema AI

Flusso di Elaborazione:

Utente → Domanda → Normalizzazione → Ricerca CSV →
├── Se match ≥ 60% → Riformulazione AI → Risposta
└── Se match < 60% → Generazione AI → Risposta generale

Installazione

⚠️ Prerequisiti
  • Node.js (versione 18 o superiore)
  • LM Studio installato sul computer
  • 8GB RAM minimo (16GB consigliato)
  • 10GB spazio disco libero

Passo 1: Setup Base

$ mkdir morato-assistenza
$ cd morato-assistenza
$ npm init -y
$ npm install express cors csv-parse string-similarity
$ npm install @lmstudio/sdk

Passo 2: Configurazione LM Studio

  1. Scarica e installa LM Studio da lmstudio.ai
  2. Avvia LM Studio e naviga in "Models"
  3. Cerca e scarica: llama-3.2-3b-instruct
  4. Vai in "Local Server" e avvia il server (porta 1234)
  5. Verifica che "Server is running" sia verde

Parametri AI Regolabili

server.js - Parametri risposta 
const result = await model.respond([
    { role: "system", content: SYSTEM_PROMPT },
    { role: "user", content: prompt }
], {
    max_tokens: 80,      // Lunghezza massima risposta
    temperature: 0.1,    // Creatività (0.0-1.0)
    top_p: 0.9,          // Nucleus sampling
    frequency_penalty: 0.1,
    presence_penalty: 0.1
});

Soglia Similarità

// Cambia soglia matching (riga ~40)
return best && best.score >= 0.6 ? records[best.index] : null;

Risoluzione Problemi

Problema: LM Studio non risponde

1. Verifica che LM Studio sia in esecuzione
2. Controlla che il modello sia scaricato e selezionato
3. Riavvia il server LM Studio
4. Verifica porta (default: 1234)

Problema: Errori CORS nel browser

1. Verifica che il middleware CORS sia installato
2. Assicurati che le richieste vengano da localhost:8080
3. Controlla gli headers nella risposta

Problema: Risposte AI non pertinenti

1. Modifica SYSTEM_PROMPT per essere più specifico
2. Aggiusta temperature a 0.1-0.3 per meno creatività
3. Aggiungi più esempi al dataset.csv

Problema: Performance lente

1. Usa modelli più piccoli (3B parametri invece di 7B)
2. Riduci max_tokens a 50-80
3. Implementa caching delle risposte frequenti
